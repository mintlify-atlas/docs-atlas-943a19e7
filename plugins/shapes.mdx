Shapes control how each particle is drawn on the canvas. Each shape is a separate npm package that implements `IShapeDrawer` and registers itself under one or more shape type names. You select a shape by setting `particles.shape.type` in your options to the registered name.

## Registering a shape

Every shape package exports a `load*Shape` function that accepts the engine instance:

```typescript
import { tsParticles } from "@tsparticles/engine";
import { loadStarShape } from "@tsparticles/shape-star";
import { loadCircleShape } from "@tsparticles/shape-circle";

await loadStarShape(tsParticles);
await loadCircleShape(tsParticles);

await tsParticles.load({
  id: "canvas",
  options: {
    particles: {
      shape: {
        type: "star", // or "circle"
        options: {
          star: {
            sides: 5,
            inset: 2,
          },
        },
      },
    },
  },
});
```

Shape-specific options live under `particles.shape.options.<shapeName>`. The key must match the registered shape type name.

## Available shapes

### circle

**Package:** `@tsparticles/shape-circle` · **Function:** `loadCircleShape` · **Type name:** `"circle"`

Draws a circular arc. Supports a configurable angle range to produce partial arcs (for example, pac-man shapes).

```typescript
import { loadCircleShape } from "@tsparticles/shape-circle";
await loadCircleShape(engine);
```

```json
{
  "particles": {
    "shape": {
      "type": "circle",
      "options": {
        "circle": {
          "angle": { "min": 0, "max": 360 }
        }
      }
    }
  }
}
```

### star

**Package:** `@tsparticles/shape-star` · **Function:** `loadStarShape` · **Type name:** `"star"`

Draws a star with a configurable number of points and inset ratio that controls how sharp or blunt the points are.

```typescript
import { loadStarShape } from "@tsparticles/shape-star";
await loadStarShape(engine);
```

```json
{
  "particles": {
    "shape": {
      "type": "star",
      "options": {
        "star": {
          "sides": 5,
          "inset": 2
        }
      }
    }
  }
}
```

The `inset` value is the ratio between the outer and inner radius of the star. Higher values produce more elongated points.

### image

**Package:** `@tsparticles/shape-image` · **Function:** `loadImageShape` · **Type names:** `"image"`, `"images"`

Renders arbitrary images (PNG, JPG, SVG, GIF) as particles. SVGs support dynamic color replacement. GIFs are animated frame by frame.

```typescript
import { loadImageShape } from "@tsparticles/shape-image";
await loadImageShape(engine);
```

```json
{
  "particles": {
    "shape": {
      "type": "image",
      "options": {
        "image": {
          "src": "https://example.com/logo.svg",
          "width": 100,
          "height": 100,
          "replaceColor": true,
          "gif": false
        }
      }
    }
  }
}
```

<Note>
  `replaceColor: true` makes the engine replace the image's fill color with the particle color. This only works for SVGs. Set `gif: true` when pointing at a GIF file.
</Note>

**Package:** `@tsparticles/shape-emoji` · **Function:** `loadEmojiShape` · **Type name:** `"emoji"`

Renders emoji characters as particles, with optional custom font and padding.

```typescript
import { loadEmojiShape } from "@tsparticles/shape-emoji";
```

## Writing a custom shape

Implement `IShapeDrawer` from `@tsparticles/engine` and register it with `engine.pluginManager.addShape`.

```typescript
import { type Engine, type IShapeDrawer, type IShapeDrawData } from "@tsparticles/engine";

class DiamondDrawer implements IShapeDrawer {
  draw({ context, drawRadius }: IShapeDrawData): void {
    context.moveTo(0, -drawRadius);
    context.lineTo(drawRadius, 0);
    context.lineTo(0, drawRadius);
    context.lineTo(-drawRadius, 0);
    context.closePath();
  }
}

export async function loadDiamondShape(engine: Engine): Promise<void> {
  await engine.pluginManager.register(e => {
    e.pluginManager.addShape(["diamond"], async () => new DiamondDrawer());
  });
}
```

Register the loader before calling `tsParticles.load()`, then use `"diamond"` as the shape type in your configuration.
