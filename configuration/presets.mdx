A preset is a pre-built tsParticles configuration registered under a name. You enable it by adding the `preset` key to your options object. You can then override any individual option to customize the preset further.

```json
{
  "preset": "snow"
}
```

Presets are distributed as separate npm packages under the `@tsparticles/preset-*` namespace. Install the preset package, load it into the engine, and reference it by name in your configuration.

---

## How to use a preset

<Steps>
  <Step title="Install the preset package">
    Each preset is an independent package. Install the one you want:

    <Tabs>
      <Tab title="npm">
        ```bash
        npm install @tsparticles/preset-snow
        ```
      </Tab>
      <Tab title="yarn">
        ```bash
        yarn add @tsparticles/preset-snow
        ```
      </Tab>
      <Tab title="pnpm">
        ```bash
        pnpm add @tsparticles/preset-snow
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Load the preset into the engine">
    Import the loader function and call it with your tsParticles engine instance before calling `tsParticles.load()`.

    ```typescript
    import { tsParticles } from "@tsparticles/engine";
    import { loadSnowPreset } from "@tsparticles/preset-snow";

    await loadSnowPreset(tsParticles);
    ```
  </Step>
  <Step title="Reference the preset by name">
    Add the `preset` key to your configuration object. The preset name matches the package suffix, in camelCase.

    ```typescript
    await tsParticles.load({
      id: "tsparticles",
      options: {
        preset: "snow",
      },
    });
    ```
  </Step>
  <Step title="Override preset options (optional)">
    Any option you provide alongside `preset` overwrites the preset's defaults. This lets you customize without writing a full configuration from scratch.

    ```json
    {
      "preset": "snow",
      "background": { "color": "#1a1a2e" },
      "particles": {
        "number": { "value": 400 }
      }
    }
    ```
  </Step>
</Steps>

## Using multiple presets

The `preset` field accepts a string or an array of strings. When an array is provided, presets are applied in order, each one merged on top of the previous.

```json
{
  "preset": ["stars", "snow"]
}
```

<Note>
  Later presets in the array override earlier ones if they set the same options. Apply the most specific preset last.
</Note>

## Available presets

### Big Circles

**Package:** `@tsparticles/preset-big-circles`  
**Preset name:** `"bigCircles"`

Large, colored circles drifting upward against a white background. Useful for decorative hero sections.

<CodeGroup>

```bash install
npm install @tsparticles/preset-big-circles
```

```typescript load
import { loadBigCirclesPreset } from "@tsparticles/preset-big-circles";
await loadBigCirclesPreset(tsParticles);
```

```json use
{ "preset": "bigCircles" }
```

</CodeGroup>
