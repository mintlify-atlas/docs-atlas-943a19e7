## Get started in three steps

<Steps>
  <Step title="Install a bundle">
    Pick the bundle that fits your needs and install it via npm, yarn, or pnpm.

    ```bash
    npm install @tsparticles/react tsparticles
    ```
  </Step>
  <Step title="Initialize and configure">
    Load the engine with your chosen bundle and pass your configuration options.

    ```typescript
    import { tsParticles } from "@tsparticles/engine";
    import { loadAll } from "@tsparticles/all";

    await loadAll(tsParticles);

    await tsParticles.load({
      id: "tsparticles",
      options: {
        particles: {
          number: { value: 80 },
          color: { value: "#ffffff" },
          links: { enable: true, color: "#ffffff" },
          move: { enable: true, speed: 2 }
        }
      }
    });
    ```
  </Step>
  <Step title="Add the container element">
    Add a `div` with the matching `id` to your HTML — tsParticles handles the rest.

    ```html
    <div id="tsparticles"></div>
    ```
  </Step>
</Steps>
